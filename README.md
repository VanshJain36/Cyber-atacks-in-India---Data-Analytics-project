
# Cyber-atacks-in-India---Data-Analytics-project
## Project Overview
This project provides a visual analysis of the cyber threat landscape across India. Using Power BI, I transformed raw security data into an interactive dashboard to uncover trends, identify high-risk regions, and analyze the most common types of cyber attacks impacting the country.

## Tools & Technologies Used
Power BI: Data visualization, interactive dashboard creation, and data modeling.

Power Query: Data cleaning and transformation.

DAX: Creating custom measures and calculated columns for deeper insights.

Data Source: Kaggle

## Key Features & Visualizations
The dashboard includes several interactive elements designed to provide a comprehensive view of the data:

Attack Type Breakdown: Categorizes the attacks (e.g., Phishing, Malware, Ransomware, DDoS) to show the most prevalent threat vectors.

Interactive Slicers: Allows users to filter the dashboard by year, state, or attack type for granular analysis.

## How to Use This Repository
Download the Data: The raw dataset can be found in the data/ folder.

View the Dashboard: * Download the Cyber_Attacks_India.pbix file.
